## 🐛 问题汇总

### 【严重】P0 - 阻塞性问题

#### 1. AI 模型 API 配置错误
- **问题:** `.env` 文件配置的是 `ANTHROPIC_API_KEY`，但代码实际使用的是 `ARK_API_KEY`（火山引擎）
- **位置:** `/apps/api/services/script_engine.py` 第 10 行
- **影响:** 所有 AI 脚本生成功能无法使用
- **当前配置:**
  ```env
  ANTHROPIC_API_KEY="sk-ant-xxx"  # ❌ 错误
  ```
- **应改为:**
  ```env
  ARK_API_KEY="your-volcengine-api-key"  # ✅ 正确
  ARK_MODEL="doubao-seed-2-0-mini-260215"
  ```

#### 2. 设置页面缺失
- **问题:** `/settings` 页面返回 404
- **影响:** 用户无法修改账户设置、API 密钥、订阅信息等
- **缺失文件:** `apps/web/app/(main)/settings/page.tsx`

#### 3. 个人信息页面缺失
- **问题:** `/profile` 页面返回 404
- **影响:** 用户无法查看/修改个人信息、头像、密码等
- **缺失文件:** `apps/web/app/(main)/profile/page.tsx`

### 【中等】P1 - 功能缺陷

#### 4. API 密钥为占位符
- **问题:** `.env` 中的 API 密钥是示例值 `sk-ant-xxx`
- **影响:** 即使修正配置也无法调用 AI 服务
- **需要:** 配置真实的火山引擎 ARK API 密钥

#### 5. 登录/注册功能未验证
- **问题:** NextAuth 配置需要验证
- **位置:** `apps/web/lib/auth.ts`
- **待确认:** 
  - 数据库用户表是否已创建
  - 邮件服务是否配置（如需要邮件验证）
  - OAuth 提供商是否配置

### 【轻微】P2 - 体验优化

#### 6. 前端 Dockerfile 缺失
- **问题:** `apps/web/Dockerfile` 是临时创建的，可能需要优化
- **建议:** 检查构建流程是否最优

#### 7. 环境变量管理
- **问题:** `.env` 文件中有硬编码的 `NEXTAUTH_SECRET`
- **建议:** 使用 `openssl rand -base64 32` 生成随机密钥
